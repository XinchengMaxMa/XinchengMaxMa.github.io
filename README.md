# Xincheng Max Ma Academic Website

This repository contains the source files for the personal academic website of Xincheng Max Ma.

## How to publish on GitHub Pages

1. Create or rename your repository to `XinchengMaxMa.github.io`.
2. Upload these files to the root of the repository:
   - `index.html`
   - `assets/styles.css`
   - `README.md`
   - `robots.txt`
   - `sitemap.xml`
3. Go to **Settings → Pages**.
4. Select **Deploy from a branch**.
5. Choose `main` and `/root`, then save.
6. Your website should appear at `https://xinchengmaxma.github.io/`.

## After publishing

- Add the website URL to Google Scholar, SSRN, LinkedIn, ORCID, HKUST profile, CV, and email signature.
- Submit the URL in Google Search Console.
- If you add a custom domain later, update the canonical URL in `index.html` and the URLs in `sitemap.xml`.
